# Parent Message Module — Detailed Specification

## 1. Module Purpose
Generate professional, polite, and clear parent communication messages.  
These messages should be short and reflect real school communication style.

---

## 2. Inputs

The module accepts:

- **Purpose of message:**  
  - Appreciation  
  - Warning  
  - Homework reminder  
  - Attendance issue  
  - Behaviour note  
  - PTM reminder  
- **Student detail:** (No real names. Use placeholders.)  
- **Action required:** (optional)

---

## 3. Output Format (Strict Markdown)

The message MUST follow this exact structure:

```markdown
**Dear Parent,**

[Main message sentence(s) — short and polite.]

**Action Required:**  
- [If needed]

**Thank you,**  
**Class Teacher**
