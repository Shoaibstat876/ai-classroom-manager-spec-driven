## ğŸ§© Modules Included

### 1. Lesson Plan Module  
Generate structured lesson plans using a predefined template.

### 2. Worksheet Module  
MCQs, short questions, fill blanks, matching, and answer keys.

### 3. Quiz & Exam Module  
MCQs, short questions, long questions, marks distribution.

### 4. Parent Message Module  
Polite, short, and professional parent communication.

### 5. Behaviour Note Module  
Neutral, factual behaviour reports with teacher actions.

### 6. Progress Summary Module  
Strengths, weaknesses, and teacher recommendations.

### 7. Timetable Module  
Weekly class timetables in clean markdown tables.

### 8. Documentation Module  
Converts outputs into Docusaurus-ready markdown pages.
