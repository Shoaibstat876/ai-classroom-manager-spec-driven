# Example 1 — Project Overview Page

# AI Classroom Manager — Project Overview

## Overview
This document provides a high-level introduction to the AI Classroom Manager Agent.  
It explains the project’s purpose, architecture, modules, and development pattern.

---

## Content

### Project Purpose
The AI Classroom Manager is designed to assist teachers by generating:

- Lesson Plans  
- Worksheets  
- Quizzes & Exams  
- Parent Messages  
- Behaviour Notes  
- Progress Summaries  
- Timetables  

All content is created using a **Spec-Driven Development** workflow.

---

### Architecture Pattern
The project follows the Spec-Kit pattern:

1. Context  
2. Constitution  
3. Specification  
4. Plan  
5. Tasks  
6. Implementation  

This ensures zero vibe coding and predictable output quality.

---

### Modules Included
- Lesson Plan Module  
- Worksheet Module  
- Exam Module  
- Parent Message Module  
- Behaviour Note Module  
- Progress Summary Module  
- Timetable Module  
- Documentation Module  

All modules have detailed specs and examples.
