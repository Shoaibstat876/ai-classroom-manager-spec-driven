# AI Classroom Manager Agent — Specification (Blueprint)

## 1. Project Overview

The **AI Classroom Manager Agent — Spec-Driven Edition** is a structured, rule-based AI assistant designed to help school teachers by generating high-quality teaching materials and documentation using Spec-Driven Development.  
All features follow the pattern:

> Specification → Plan → Tasks → Implementation

This ensures no vibe coding and 100% controlled output.

---

## 2. Core Capabilities (Modules)

The system consists of the following major modules:

1. **Lesson Plan Generator**  
2. **Worksheet Generator**  
3. **Quiz & Exam Generator**  
4. **Parent Message Generator**  
5. **Behaviour Note Generator**  
6. **Progress Summary Generator**  
7. **Timetable & Schedule Generator**  
8. **Documentation Generator (Docusaurus-ready)**  

Every module has:  
- Inputs  
- Processing rules  
- Output format  
- Constraints
